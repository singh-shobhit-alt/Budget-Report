# Fix 404 Error on GitHub Pages

## Quick Fix Steps

### Step 1: Find Your Repository Name
1. Go to your GitHub repository
2. Look at the URL: `https://github.com/username/REPO-NAME`
3. The repository name is the last part (e.g., `Budget-Report`)

### Step 2: Update vite.config.js

Open `vite.config.js` and find this line:
```javascript
const REPO_NAME = 'Budget-Report'; // ← CHANGE THIS
```

**Change it to match your repository name exactly:**
- If your repo is `Budget-Report` → keep `'Budget-Report'`
- If your repo is `budget-report` → use `'budget-report'` (case-sensitive!)
- If your repo is `my-budget-app` → use `'my-budget-app'`

### Step 3: Rebuild and Deploy

1. **Rebuild the project:**
   ```bash
   npm run build
   ```

2. **If using GitHub Actions:**
   - Commit and push the changes
   - The workflow will automatically rebuild and deploy

3. **If deploying manually:**
   - Go to repository Settings → Pages
   - Make sure the source is set correctly
   - The `dist` folder should be deployed

### Step 4: Verify

1. Wait 1-2 minutes for GitHub Pages to update
2. Visit your GitHub Pages URL
3. Check browser console (F12) - should see no 404 errors

## Common Issues

### Issue: Still getting 404
**Solution:** 
- Make sure the base path has a trailing slash: `/Repo-Name/` (not `/Repo-Name`)
- Make sure the repository name matches exactly (case-sensitive)
- Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)

### Issue: Works locally but not on GitHub Pages
**Solution:**
- The base path is only used in production builds
- Make sure you're building with `npm run build`
- Check that the `dist` folder has the correct structure

### Issue: Assets not loading
**Solution:**
- Check Network tab in browser DevTools
- Look for which files are 404
- Verify the base path matches your repo name

## Example Configurations

**For repository: `username/Budget-Report`**
```javascript
const REPO_NAME = 'Budget-Report';
// Base will be: '/Budget-Report/'
```

**For repository: `username/budget-report`**
```javascript
const REPO_NAME = 'budget-report';
// Base will be: '/budget-report/'
```

**For user pages: `username/username.github.io`**
```javascript
const REPO_NAME = ''; // Empty string
// Base will be: '/'
```
